會計與預算系統

📊 專案簡介

這是一個功能完整的個人記帳與預算管理系統，旨在幫助使用者有效管理個人財務、追蹤支出、設定預算目標，並透過視覺化報表深入了解財務狀況。

本專案是個人全端開發作品，展示了現代 Web 應用程式開發的完整流程，從前端介面設計到後端 API 開發，再到資料庫設計與部署。

✨ 主要功能

  • 多幣別支援：支援 TWD、USD、JPY 等多種貨幣，並提供即時匯率轉換
  • 交易管理：快速記帳、交易分類、描述自動完成
  • 預算追蹤：設定月度/日度預算，即時追蹤預算使用狀況
  • 報表分析：多維度財務報表，包含類別分析、帳戶分析、排名統計等
  • 安全性：使用者認證、JWT Token、2FA 雙因素驗證

🛠️ 技術堆疊

前端

  • Vue 3 (Composition API)
  • TypeScript
  • Pinia (狀態管理)

後端

  • Python FastAPI
  • SQLAlchemy (ORM)
  • PostgreSQL

部署

  • Docker & Docker Compose
  • Nginx (反向代理)
  • SSL/TLS (HTTPS)
